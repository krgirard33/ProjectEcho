Project Echo: User Guide

Project Echo is a simple web-based application designed to help you quickly log activities, track projects, and manage your to-do list.

Someone showed me Interstitial Journaling, and I thought it looked interesting, as I needed a a better way to track what I am working on at work. For more information see https://nesslabs.com/interstitial-journaling

1. Home Page (Journal Log)

The main page (/) is your dashboard for logging new entries and reviewing recent activity.

New Entry Fields

Element Purpose
Project (Input) Organizes your entry. Optional.
Text Area Logs your activity and thoughts.

Past Entries (Last 14 Days)

Element Action Result
Date Header (e.g., 2025-10-05) Click on the date. The entries for that day will toggle (show/hide).
Entry Link Click on the entry content. Takes you to the Edit page for that specific entry.

Formatting Entries (Markdown-like Syntax)

Syntax Example Input Result in Display
New Line Line 1.\nLine 2.

Line 1.
Line 2 also works.		 Renders on two separate lines.
Unordered List * Item A\n* Item B

* Line 1.
* Line 2 also works.		 Creates a standard bulleted (<ul>) list.
Ordered List # Step 1\n# Step 2

# Line 1.
# Line 2 also works.		 Creates a numbered (<ol>) list.

2. Calendar View

The Calendar view (/calendar) lets you navigate months and review your journal entries by day.

Element ActionResult
< Previous / Next > Click the navigation buttons. Changes the displayed month and year.
Day Number Click a day number. Takes you to the Day View (green highlight indicates entries exist).

Editing Entries

In Day View, clicking any entry takes you to the **Edit Entry** page where you can modify the Project and Entry Content fields.

3. To-Do List

The To-Do List page (/todo) is for managing your tasks, organized by **Project**.

Task Management

Section Description Action
Active Tasks Tasks with status: backlog, holding, or started. Grouped by project. Click ✏️ to edit priority, dates, or mark as *finished*.
Done Tasks Tasks with status: finished. Grouped by project and crossed out. Click ✏️ to view details or change status back to active.
